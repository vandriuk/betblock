## Наступні кроки (по пріоритету)

### Пріоритет 1 — Запуск у прод

#### 1.1 Кастомний домен
- Підключити свій домен замість `shlakoblock-ebf35.web.app`
- Firebase Console → Hosting → Custom domain
- Потрібно: домен + доступ до DNS
- Складність: низька (30 хв)

#### 1.2 Управління юзерами (адмін-панель)
- Створення/видалення/зміна ролі працівників прямо з додатку
- Зараз тільки через Firebase Console
- Нова вкладка "Юзери" (тільки для admin)
- Firebase Admin SDK або Cloud Functions для створення юзерів
- Складність: середня

#### 1.3 Офлайн-режим з синхронізацією
- Firestore persistence — працює без інтернету
- Автоматична синхронізація при підключенні
- Індикатор статусу з'єднання в хедері
- Складність: низька (Firestore має вбудовану підтримку)

### Пріоритет 2 — Покращення функціоналу

#### 2.1 Звіти по періодах
- Фільтрація за датами (від-до) на всіх сторінках
- Зведений звіт за місяць/квартал
- Генерація PDF звіту
- Графіки по місяцях (тренди продажів, виробництва, витрат)
- Складність: середня

#### 2.2 Фото в записах
- Прикріплення фото до замовлень/виробництва/складу
- Firebase Storage (5GB безкоштовно)
- Стиснення фото на клієнті перед завантаженням
- Складність: середня

#### 2.3 Push-нотифікації
- Повідомлення працівникам про нові замовлення
- Адміну — про низькі запаси на складі
- Firebase Cloud Messaging (FCM) — безкоштовно
- Потрібно: Cloud Functions для тригерів
- Складність: середня-висока

### Пріоритет 3 — Масштабування

#### 3.1 Історія змін (audit log)
- Хто що змінив і коли
- Firestore subcollection `{entity}/{id}/history`
- Перегляд історії для кожного запису
- Складність: середня

#### 3.2 Мультимовність (i18n)
- Підтримка інших мов крім української
- Потрібно якщо масштабувати за межі України
- Бібліотека: react-intl або i18next
- Складність: середня

#### 3.3 Друк документів
- Друк накладних, рахунків, актів
- Генерація PDF з даних замовлень/продажів
- Шаблони документів
- Складність: середня

#### 3.4 Інтеграція з бухгалтерією
- Експорт у формат 1С або BAS
- Синхронізація витрат/продажів
- Складність: висока
