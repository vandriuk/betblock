## 📊 Нові можливості

### 1. Залишки по продукції
Дашборд показує для КОЖНОГО типу:
- Вироблено всього
- Продано всього
- **На складі зараз**
- В замовленнях

### 2. Управління продукцією
Розділ "Продукція":
- Додати новий тип (Напр: "Газоблок", "Керамоблок")
- **Редагувати назву та ціну** (кнопка "олівець")
- Видалити

### 3. Розширений склад
Клік на "олівець" → редагуйте:
- ✅ Назву
- ✅ Кількість
- ✅ Одиницю виміру
- ✅ Мін. кількість
- ✅ Ціну
- ✅ **Примітки** (нове!)

### 4. Замовлення
Новий розділ для роботи з клієнтами:
- Створюйте замовлення
- Статуси: Нове → В роботі → Готово → Виконано
- Примітки до замовлення
- Відстежуйте на дашборді

### 5. Експорт та бекап
Дві кнопки вгорі:
- **🟢 Excel** - всі дані в один файл .xlsx
- **🟣 Бекап** - повний бекап .json

## 🎮 Демо vs Firebase

### ЗАРАЗ (Демо-режим):
- ✅ Працює відразу
- ✅ Всі функції доступні
- ❌ Дані тільки на вашому комп'ютері
- ❌ Немає синхронізації

### ПІСЛЯ налаштування Firebase:
- ✅ Синхронізація між пристроями
- ✅ Доступ з телефону, планшету, комп'ютера
- ✅ Багато користувачів одночасно
- ✅ Автоматичний бекап

## 📝 Як налаштувати Firebase?

**Короткий шлях (15-20 хв):**
1. Відкрийте `ІНСТРУКЦІЯ-V2.md`
2. Слідуйте Кроки 1-8
3. Замініть `firebaseConfig` в коді
4. Готово!

**Детально:**
- КРОК 1-2: Створити проект + Authentication (5 хв)
- КРОК 3-4: Firestore + правила (5 хв)
- КРОК 5-7: Web App + користувачі (5 хв)
- КРОК 8-9: Оновити код + Vercel (5 хв)

## 💡 Поради

### Як додати новий тип продукції?
1. Розділ "Продукція"
2. "+ Додати продукцію"
3. Назва, ціна → Додати

### Як змінити ціну продукції?
1. Розділ "Продукція"
2. Кнопка "олівець" біля продукції
3. Змініть назву або ціну
4. Кнопка "✓ Зберегти"

### Як редагувати матеріал на складі?
1. Розділ "Склад"
2. Кнопка "олівець" біля матеріалу
3. Змініть дані
4. Кнопка "✓"

### Як створити замовлення?
1. Розділ "Замовлення"
2. Заповніть форму (клієнт, продукція, кількість)
3. Додати замовлення
4. Змінюйте статус у списку

### Як зробити експорт?
1. Кнопка "Excel" вгорі справа
2. Файл автоматично завантажиться
3. Відкрийте у Excel

## ⚠️ Важливо знати
